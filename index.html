Chat Stats Dashboard

Локальный дашборд для отслеживания активности по участникам, дням и последним сообщениям. Можно загрузить JSON с данными, либо подхватывать файл автоматически из stats.json.

Источник данных

Загрузи JSON-файл или вставь данные вручную.

Формат поддерживает поля: participants, days, updatedAt. Данные берутся из общего файла stats.json за последние 30 дней. Если есть subscribers.json, дашборд показывает их даже когда в чате нет свежих сообщений.
